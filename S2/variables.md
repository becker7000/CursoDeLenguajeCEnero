## Variables

*Las variables son contenedores de datos.*

**Naturalmente un programa se compone de 3 partes:**

1. Entradas/Salidas
2. Calculos o procesos
3. Estructuras de control

### Ejemplo: 

*Entradas:*
```
   a = 10
   b = 20
   c = 5
``

*Calculos:*
```
   a = a + 3 = 10 + 3 = 13
   b = b + 4 - a = 20 + 4 - 13 = 11
   c = a + b + c = 13 + 11 + 5 = 29
   a = a + c = 13 + 29 = 42
   b = 4
   c = c + 3 - b + 2 = 29 + 3 - 4 + 2 = 30 
```

*Salidas:*
```
    a = 42
    b = 4
    c = 30
```
